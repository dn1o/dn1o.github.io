GitHub

Napravi GitHub racun, zatim klikni na "New Repository". Daj joj ime: tvojGithubusername.github.io. Zameni tvojGithubusername sa imenom kojim si se registrovao na github.
Klikni na "Create Repository", izaci ce ekran sa dva text boxa. Ostavi GitHub upaljen i predji na sledeci korak.

Cloud9

Idi na sajt Cloud9 i uloguj se preko GitHub-a.
Napravi novi workspace sa bilo kojim imenom i opisom. Odaberi html5 kao opciju. Kad se workspace ucita, vrati se tamo gde si stao sa GitHubom. Ne gasi workspace.
Iskopiraj tekst iz prve kutije ali izostavi prvu liniju koja pocinje sa echo. Vrati se na tab u kome je workspace i pejstuj u terminal ono sto si malopre iskopirao.
Terminal je dole, pocetak unosa je obelezen znakom $. Pritisni Enter.

Napravi i objavi sajt

Napravi novi fajl u workspace-u i daj mu ime index.html. Ukucaj u novom fajlu: html:5 i pritisni tab. Izmedju body tagova napisi h1 i pritisni tab.
Izmedju h1 taga napisi "Ce Pobedimo!".
U terminalu otkucaj: git add -A && git commit -m "Napravio sam" && git push origin pritisni enter. Sajt je na internetu. Idi na adresu koja je ista kao ime repozitorije koju si napravio, i pogledaj sta si uradio. Primer: dn1o.github.io, gde je dn1o ime koje sam ja koristio kad sam se registrovao na GitHub-u.
